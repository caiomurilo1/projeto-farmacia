Projeto - Farmácia SiMedique

Sistema Desktop desenvolvido em C# utilizando Windows Forms (.NET Framework), com aplicação dos conceitos de Programação Orientada a Objetos (POO).

Objetivo

Desenvolver uma aplicação para cadastro de medicamentos em uma farmácia, aplicando os conhecimentos adquiridos em sala de aula, com foco em organização estrutural e modelagem orientada a objetos.

Estrutura do Sistema

O sistema é composto por três telas:

1. Tela Principal (Menu)
Tela inicial apresentada ao usuário, contendo o nome do software, nome dos participantes, imagem de fundo com o tema da aplicação e menu de navegação para acessar o cadastro de medicamentos.

2. Tela de Medicamentos (Cadastro e Listagem)
Tela responsável pelo cadastro de medicamentos e exibição dos registros cadastrados.

3. Tela de Visualização de Medicamentos
Tela destinada à exibição completa dos dados do medicamento selecionado, com campos configurados como somente leitura.


Tecnologias Utilizadas

* C#
* Windows Forms (.NET Framework)
* Visual Studio
