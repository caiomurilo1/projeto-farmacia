# Farmácia SiMedique

Sistema Desktop desenvolvido em C# utilizando Windows Forms (.NET Framework).

## Objetivo

Desenvolver uma aplicação para cadastro de medicamentos.

## Estrutura do Sistema

O sistema é composto por três telas:

1. Tela Principal (Menu)
2. Tela de Medicamentos
3. Tela de Visualização

## Tecnologias Utilizadas

- C#
- Windows Forms (.NET Framework)
- Visual Studio
